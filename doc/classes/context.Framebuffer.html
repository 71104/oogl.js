context.Framebuffer Class

Defined in: src\Framebuffer.js:7
Module: context

Wraps a GL framebuffer object.

Instancing an object of this class is equivalent to calling the GL function createFramebuffer. The returned WebGLFramebuffer object is extended by OOGL-specific features and returned by the Framebuffer constructor.

getReflected

(
  • n
)
OOGL.Vector3

Defined in src\Vector3.js:323

Computes the reflection of this vector against a line whose normal vector is specified. The reflection of a vector v is computed as:

v - 2 * (v.n) * n

where v.n is the dot product between the vector and the normal.

The computed vector is returned as a new Vector3 object, this vector is not changed.

Parameters:

  • n OOGL.Vector3

    The normal vector.

Returns:

OOGL.Vector3: The computed reflected vector.

Example:

// reflects a velocity vector against a floor
var newVelocity = velocity.getReflected(new OOGL.Vector3(0, 1, 0));

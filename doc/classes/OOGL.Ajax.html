getJSON

  • url
  • [callback]
)

Defined in src\Ajax.js:251

Performs a GET AJAX request. The data returned from the server is parsed as JSON and passed to a user-defined callback function.

Parameters:

  • url String

    url The URL to request.

  • [callback] Function optional

    An optional one-argument user-defined callback function that is invoked when the request completes successfully.

Example:

OOGL.Ajax.getJSON('meshes/box.json', function (box) {
    vertices = new oogl.VertexArray(0, 3, box.vertices);
    textureCoordinates = new oogl.VertexArray(1, 2, box.textureCoordinates);
    // ...
});

Lets the user define a callback function that gets called when an error related to an AJAX request occurs.

Parameters:

  • callback Function

    A user-defined callback function that gets called in case of an error in an AJAX request.

Example:

OOGL.Ajax.onError(function () {
    alert('AJAX error occurred.');
});

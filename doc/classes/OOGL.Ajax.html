Defined in src\Ajax.js:62

Performs a GET AJAX request. The data returned from the server is passed to a user-defined callback function.

Parameters:

  • url String

    The URL to request.

  • callback Function

    A one-argument user-defined callback function that is invoked when the request completes successfully.

Example:

OOGL.Ajax.get('shaders/frag/box.frag', function (source) {
    fragmentShader = new oogl.FragmentShader(source);
    // ...
});

    The URL to request.

Example:

    vertices = new oogl.VertexArray(0, 3, box.vertices);
    textureCoordinates = new oogl.VertexArray(1, 2, box.textureCoordinates);
    // ...
});

context.Renderbuffer Class

Defined in: src\Renderbuffer.js:7
Module: context

Wraps a GL renderbuffer.

Instancing an object of this class is equivalent to calling the GL function createRenderbuffer. The returned WebGLRenderbuffer object is extended by OOGL-specific features and returned by the Renderbuffer constructor.

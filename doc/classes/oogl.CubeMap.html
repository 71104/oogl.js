image2D

(

  • format
  • type
  • object
)

Inherited from oogl.Texture: src\Textures.js:231

Specifies an image, canvas or video for this texture.

gl.texImage2D equivalent.

Parameters:

  • level Number

    The mipmap reduction level.

  • format Number

    The texel format; can be gl.ALPHA, gl.RGB, gl.RGBA, gl.LUMINANCE or gl.LUMINANCE_ALPHA.

  • type Number

    The binary data type; can be gl.UNSIGNED_BYTE, gl.UNSIGNED_SHORT_5_6_5, gl.UNSIGNED_SHORT_4_4_4_4 or gl.UNSIGNED_SHORT_5_5_5_1.

  • object Mixed

    A DOM image, canvas or video element to use as texture image.

Example:

texture.image2D(0, oogl.RGBA, oogl.UNSIGNED_BYTE, image);

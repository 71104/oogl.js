() Number[]

Defined in src\Vector4.js:51

Returns this vector as an array of four elements.

Returns:

Number[]: An array containing the X, Y, Z and W components of this vector.

Example:

var v = new OOGL.Vector4(1, 2, 3, 1);
program.uniform4fv('Position', v.toArray());

toStandard

() OOGL.Vector3

Defined in src\Vector4.js:65

Converts this homogeneous vector to a 3-component standard vector dividing the X, Y and Z components by the W component. This method produces a new OOGL.Vector3 object, while this vector is not changed.

Returns:

OOGL.Vector3: The computed standard vector.

Example:

var w = new OOGL.Vector4(2, 4, 6, 2);
var v = w.toStandard(); // v is (1, 2, 3)

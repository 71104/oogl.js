OOGL.Matrix2 Class

Defined in: src\Matrix2.js:7
Module: OOGL

A mutable 2x2 matrix.

Constructor

OOGL.Matrix2

(
  • data
)

Defined in src\Matrix2.js:7

Parameters:

  • data Number[]

    A 4-element array of the floating point values to be put into the matrix.

    Matrix elements are specified in column-major order.

    The specified data array is duplicated into the matrix, changes to it will not affect the content of the matrix.

    An exception is thrown if the length of the data array is not 4.

Example:

var matrix = new OOGL.Matrix2([1, 0, 0, 1]); // creates the 2x2 identity matrix

Defined in src\Matrix2.js:85

Returns the element at the specified row and column in the matrix.

Row and column indices are zero-based. This method is equivalent to fetching the j * 2 + i-th element of the array:

matrix.get(i, j) == matrix[j * 2 + i] // true

Parameters:

  • i Number

    The row index.

  • j Number

    The column index.

Returns:

Number:

The value at the specified row and column.

Example:

var m = new OOGL.Matrix2([1, 2, 3, 4]);
var determinant = m.get(0, 0) * m.get(1, 1) - m.get(0, 1) * m.get(1, 0); // -5

Defined in src\Matrix2.js:105

Changes the element at the specified row and column in the matrix.

Row and column indices are zero-based. This method is equivalent to setting the j * 2 + i-th element of the array:

matrix.put(i, j, x);
matrix[j * 2 + i] = x; // same as previous

Parameters:

  • i Number

    The row index.

  • j Number

    The column index.

  • value Number

    The value to put at the specified row and column.

Example:

var matrix = new OOGL.Matrix2([3, 3, 0, 3]);
matrix.put(0, 1, 3); // now matrix is [3, 3, 3, 3]
